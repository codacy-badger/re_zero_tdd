# FizzBuzz

## 仕様
+ ３で割り切れる場合はFizzを返す
+ 5で割り切れる場合はBuzzを返す
+ 両方で割り切れる場合はFizzBuzzを返す
+ それ以外では値を返す
+ 繰り返し実行できるようにする
